1 Dados coletados

O VenderCerto foi desenvolvido com privacidade como prioridade. Não exigimos cadastro, login ou e-mail para usar o aplicativo.

Dados inseridos voluntariamente pelo usuário

🔒 Armazenado apenas no dispositivo
  • Nome da empresa, CNPJ e regime tributário (opcional no onboarding)
  • Alíquota de imposto e custos fixos mensais
  • Dados de DRE: faturamento, CMV, comissões e impostos por mês
  • Dados de Fluxo de Caixa: entradas e saídas mensais
  • Produtos favoritos salvos (nome, preço, custo, margem)
  • Histórico de cálculos de marketplace, loja física e delivery

Dados gerados automaticamente pelo app

🔒 Armazenado apenas no dispositivo
  • Preferências de tema (claro/escuro)
  • Status de onboarding concluído
  • Data da última confirmação de taxas
  • Setores de operação configurados

⚠️ Nenhum dado pessoal é enviado a servidores externos. O VenderCerto funciona 100% offline. Todos os dados ficam no armazenamento local do seu dispositivo (AsyncStorage) e nunca saem dele — salvo nas situações de terceiros descritas abaixo.

2 Como usamos os dados

Os dados inseridos no app são utilizados exclusivamente para:

  • Calcular margens, lucros e preços nos módulos do app
  • Ratear custos fixos automaticamente em cada cálculo
  • Gerar DRE e Fluxo de Caixa mensais
  • Salvar e recarregar produtos favoritos
  • Manter preferências de configuração entre sessões

Não utilizamos seus dados para publicidade, analytics externos, perfis de uso ou qualquer finalidade além das listadas acima.

3 Armazenamento e segurança

Todos os dados são armazenados localmente no dispositivo usando AsyncStorage (React Native). Isso significa:

  • Os dados ficam no seu aparelho e não são transmitidos pela internet
  • Desinstalar o app apaga todos os dados automaticamente
  • Não há backup automático em nuvem — faça backup manual se necessário
  • Não temos acesso aos dados armazenados no seu dispositivo

Imagem de compartilhamento

Ao usar a função "Compartilhar resultado", o app gera uma imagem localmente no dispositivo. Essa imagem é compartilhada pelo sistema operacional — não passa por nossos servidores.

4 Serviços de terceiros

Google Play (distribuição)

O app é distribuído pela Google Play Store. A Google pode coletar dados de uso agregados conforme sua Política de Privacidade.

Expo / React Native (infraestrutura)

O app é construído com Expo. Dados de crash e erros podem ser coletados anonimamente conforme os termos do Expo.

📢 Sem rastreamento de anúncios. O VenderCerto não contém SDKs de publicidade, pixels de rastreamento, Facebook SDK ou qualquer tecnologia de monitoramento comportamental.

5 Funcionalidades Premium

⭐ Modelo Freemium — leia com atenção

O VenderCerto atualmente oferece todas as funcionalidades gratuitamente. No futuro, alguns módulos poderão se tornar pagos. A qualquer momento e sem aviso prévio obrigatório, funcionalidades gratuitas podem migrar para o plano pago. Notificaremos os usuários pelo próprio app com a maior antecedência possível.

Quando o modelo de assinatura for ativado, utilizaremos:

  • RevenueCat — gerenciamento de assinaturas. Coleta identificador anônimo do dispositivo e histórico de compras.
  • Google Play Billing — processamento de pagamentos. Sujeito à Política do Google.

Dados de pagamento nunca são armazenados pelo VenderCerto.

6 Seus direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Acesso: consultar quais dados estão armazenados (acessíveis dentro do próprio app)
  • Correção: editar seus dados diretamente em Minha Empresa e nos módulos
  • Exclusão: apagar todos os dados desinstalando o app ou usando "Redefinir dados" nas Configurações
  • Portabilidade: exportar todos os seus dados em formato JSON através de Configurações → Exportar meus dados. O arquivo inclui dados de empresa, DRE, Fluxo de Caixa, favoritos e configurações.
  • Revogação: revogar qualquer consentimento a qualquer momento

7 Menores de idade

O VenderCerto é destinado a pessoas maiores de 18 anos. Não coletamos intencionalmente dados de menores de idade.

8 Atualizações desta política

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas através do app na tela "O que há de novo". O uso continuado do VenderCerto após mudanças constitui aceite dos novos termos.

1 O Serviço

O VenderCerto é um aplicativo mobile de calculadora de precificação e análise de margem para vendedores brasileiros. O aplicativo oferece:

  • Calculadora de margem para marketplaces (ML, Shopee, Amazon, Magalu)
  • Calculadora de margem para loja física e delivery
  • Comparador de canais de venda
  • Precificador reverso (calcula preço pela margem desejada)
  • DRE simplificado e Fluxo de Caixa mensal
  • Histórico de favoritos e cálculos salvos
  • Conteúdo educacional sobre precificação

⚠️ Os resultados apresentados são estimativas. O VenderCerto utiliza tabelas de taxas que podem estar desatualizadas. Sempre confirme as taxas diretamente nas plataformas antes de tomar decisões comerciais. O app não substitui consultoria contábil ou financeira profissional.
